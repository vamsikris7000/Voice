Basic Integration

Add this script tag to your website to embed the chat widget:

<script src="https://your-domain.com/widget/embed.js" data-backend-url="http://localhost:3001" data-position="bottom-right" data-primary-color="#667eea" data-auto-open="false"> </script>

Configuration Options

Available configuration attributes:

data-backend-url="http://localhost:3001" // Required: Your backend URL data-position="bottom-right" // Widget position data-primary-color="#667eea" // Primary color data-secondary-color="#f7fafc" // Secondary color data-text-color="#2d3748" // Text color data-auto-open="false" // Auto-open on page load data-welcome-message="Hello!" // Custom welcome message data-api-key="your-api-key" // Optional API key data-user-id="custom-user-id" // Custom user ID data-enable-escalation="true" // Enable human escalation data-escalation-keywords="human,agent" // Escalation trigger words data-typing-indicator="true" // Show typing indicator data-sound="false" // Enable sound notifications data-max-messages="100" // Max messages to keep data-widget-id="my-chat-widget" // Custom widget ID

Advanced Usage

For more control, you can initialize the widget programmatically:

<script> const widget = new BidirectionalChatWidget({ backendUrl: 'http://localhost:3001', position: 'bottom-left', primaryColor: '#e53e3e', onMessage: (message) => { console.log('New message:', message); }, onEscalation: (info) => { console.log('Escalated to human agent:', info); }, onError: (error) => { console.error('Widget error:', error); } }); </script>

‚öôÔ∏è Configuration Options

  • data-backend-url - Your backend URL (required if not using Dify)
  • data-dify-api-url - Your Dify API URL (required if not using backend)
  • data-dify-api-key - Your Dify API key (required if not using backend)
  • data-position - Widget position: bottom-right, bottom-left, top-right, top-left, bottom-center
  • data-primary-color - Primary color for the widget (hex color)
  • data-auto-open - Auto-open widget on page load (true/false)
  • data-theme - Widget theme: light, dark
